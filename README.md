## 配置文件

* 添加编译配置文件

1. New File->Other->Configuration Settings File
2. PROJECT->Info->Configurations->Debug/Relaese->选择对应的config文件
3. TARGETS->Build Settings->OC(User-Defined)/Swift(Swift Compile-Custom Flags)中设置自定义的键值
